This telegram bot checks the price of games from the Steam that you add to the your list.
You can send an ID of a game or a link to the Steam page.
By default it check the price every day at 21:00 UTC+2.

Buttons:
Get List - return all games that you add to your list one by one.
Add Game - add a new game to the your list, you can send an ID of a game or a link to the Steam.
Delete game - delete the game from your list, can get only IDs.(also you can delete the game after you request Get List - there is a button below each game).
